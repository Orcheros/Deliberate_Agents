# Video Format Templates

## Short (≤60 seconds) — Vertical 9:16

```
HOOK [0:00-0:03]
  - Visual: [attention grab]
  - Spoken: [ONE sentence that hooks]
  - Text: [2-4 words on screen]

BODY [0:03-0:45]
  - 3-5 segments, 8-10 seconds each
  - Each segment: one point + one visual change
  - Pattern interrupt at midpoint (~0:20)

PAYOFF [0:45-0:55]
  - Deliver the promise from the hook
  - Visual: [satisfying conclusion]

CTA [0:55-0:60]
  - One action (follow, save, check link)
  - Keep it natural, not begging
```

## Standard (3-5 minutes) — Horizontal 16:9

```
COLD OPEN [0:00-0:10]
  - Hook: most interesting claim or moment
  - No intro, no greeting

CONTEXT [0:10-0:45]
  - Why this matters / who it's for
  - Set expectations: "By the end of this..."

CHAPTER 1 [0:45-2:00]
  - Main point #1
  - Example/proof/demonstration
  - Transition to next chapter

CHAPTER 2 [2:00-3:15]
  - Main point #2
  - Example/proof/demonstration
  - Transition

CTA [3:15-3:30]
  - Placed at ~70% (not the end)
  - Earned by value already delivered

CHAPTER 3 / CONCLUSION [3:30-4:30]
  - Final point or synthesis
  - Restate key takeaway

OUTRO [4:30-5:00]
  - What to watch next
  - Brief sign-off
```

## Long (10-15 minutes) — Horizontal 16:9

```
TEASER [0:00-0:15]
  - Cold open: most dramatic moment
  - "But first, let me explain..."

INTRO [0:15-1:00]
  - Context and stakes
  - Promise: "Here's what you'll learn"
  - Agenda preview (chapter markers)

CHAPTER 1 [1:00-4:00]
  - Deep dive: problem/context
  - B-roll variety: screen, face, graphics
  - Sub-points with visual markers

CHAPTER 2 [4:00-7:00]
  - Solution/framework/analysis
  - Practical examples
  - Mid-video engagement hook

MID-ROLL CTA [7:00-7:30]
  - Natural pause point
  - Subscribe/like (earned by this point)

CHAPTER 3 [7:30-11:00]
  - Application/advanced concepts
  - Case study or demonstration
  - Audience-specific branching

CONCLUSION [11:00-13:00]
  - Synthesis of all chapters
  - Key takeaways (numbered)
  - Acknowledge what wasn't covered

NEXT STEPS [13:00-14:00]
  - What to do with this info
  - Related video recommendation
  - Final CTA (community, course, newsletter)

OUTRO [14:00-15:00]
  - Personal sign-off
  - Tease next video topic
```
