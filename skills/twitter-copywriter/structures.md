# Twitter Post Structures

## 1. Single Tweet — Standalone Insight
```
[Hook/Insight in ≤280 chars]
```
Use: Quick thoughts, reactions, quotable observations

## 2. Thread — Problem → Solution
```
Tweet 1: [Problem statement — hook]
Tweet 2: [Why this matters / who it affects]
Tweet 3-N: [Solution steps, one per tweet]
Final: [Synthesis + CTA]
```
Use: Tactical advice, how-tos, frameworks

## 3. Thread — Story Arc
```
Tweet 1: [Outcome/result that hooks]
Tweet 2: [Context — where it started]
Tweet 3-N: [Journey/steps/failures]
Final: [Lesson + what's next]
```
Use: Case studies, personal experiences, lessons

## 4. Thread — List/Collection
```
Tweet 1: [Category + count — "7 tools that..."]
Tweet 2-N: [One item per tweet with brief explanation]
Final: [Meta-insight or recommendation]
```
Use: Resources, recommendations, observations

## 5. Quote Tweet — Commentary
```
[Original context] + [Your unique angle/addition/disagreement]
```
Use: News reactions, trend commentary, building on others' ideas

## 6. Poll Format
```
[Framing question that reveals something interesting]
□ Option A
□ Option B
□ Option C
□ Option D (often humorous)
```
Use: Engagement, market research, starting conversations
