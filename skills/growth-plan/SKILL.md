# Step 3: Execution Planning

## Objective

Translate growth strategy into actionable plans that downstream agents (Content Writer, SEO Specialist, SDR) can execute.

## Instructions

1. **Prioritize initiatives**:
   - Rank growth initiatives by: expected impact, effort, learning value
   - For alpha stage: bias toward fast-learning experiments over big bets
   - Select top 3-5 initiatives for the current cycle

2. **Design experiments**:
   - For each initiative:
     - **Hypothesis**: "If we [action], then [metric] will [change] because [reason]"
     - **Success criteria**: Specific metric threshold that validates the hypothesis
     - **Timeline**: How long to run before evaluating
     - **Minimum viable test**: Smallest version that generates signal

3. **Create execution briefs**:
   - **Content briefs** (for Content Writer):
     - Topic, target audience, format, key messages, CTA
     - SEO keywords to target (coordinate with SEO Specialist)
     - Distribution plan
   - **Campaign briefs** (for SDR / email):
     - Target segment, messaging angle, sequence structure
     - Personalization variables
     - Success metrics
   - **Landing page briefs** (for Developer or Designer):
     - Value proposition, social proof, CTA
     - A/B test variants if applicable

4. **Define measurement plan**:
   - What metrics to track for each initiative
   - How to attribute results
   - When to evaluate and make go/no-go decisions
   - What "failure" looks like (and what we learn from it)

5. **Write the plan** to `.deliberate/reports/{slug}/growth-plan.md`

## Output

- Prioritized initiative list with experiment designs
- Execution briefs for downstream agents
- Measurement plan with success criteria

## Transition

Plan complete. Downstream agents receive their briefs through the PjM assignment process.
